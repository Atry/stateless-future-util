com.qifun.statelessFuture.util.io

SocketWritingQueue

trait SocketWritingQueue extends AnyRef

写入Socket的缓冲数据队列。

Note

本SocketWritingQueue保证线程安全，允许多个线程向这里提交要写入的数据。 enqueue提交的缓冲区将按提交顺序排队写入，而不会混合。
